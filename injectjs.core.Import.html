(static) Import#module(sRequiredAlias) → {Promise}

Requires the module defined by the given Alias, so when requiring custom dependencies only the Alias or at least part of it needs to be provided. Given that you maped: Import.mapModulePath('my.test.alias','src/files'); When you import: Import.module('my.test.alias.with.my.test.Object'); Then you actually require: /src/files/with/my/test/Object.js > Note that '/with/my/test/' path was automatically resolved, since it was not specified. > Note that by default a '.js' will be appended by the end of the required depency.
